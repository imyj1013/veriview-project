# Veriview AI Debate Server

AI 기반 토론 시스템 서버로, Flask를 사용해 토론 세션 관리, 영상 분석, AI 응답 생성 기능을 제공합니다.

## Prerequisites
- Python 3.8+
- Git
- FFmpeg (영상 처리에 필요)
- Ollama (AI 모델 실행, http://localhost:11434)
